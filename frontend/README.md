# Enterprise AI Hiring Copilot

An AI-powered recruitment platform that automates resume screening, candidate ranking, skill gap analysis, candidate comparison, and recruiter report generation.

## Features

* Resume PDF Parsing
* Semantic Resume–Job Description Matching
* Skill Gap Analysis
* AI Recruiter Evaluation
* Candidate Ranking Dashboard
* Candidate Comparison Engine
* Recruiter PDF Report Generation
* Interactive Analytics Dashboard
* Architecture Visualization Page
* FastAPI Backend
* Next.js Frontend
