# 🚀 AI Resume Screening System

An AI-powered recruitment platform that automatically evaluates resumes against job descriptions using semantic matching, skill-gap analysis, candidate ranking, and recruiter-focused insights.

## 🌐 Live Demo

**Frontend:** Deployment
ai-resume-screening-system-mdej-bpuv3k9pa-srikan748s-projects.vercel.app

**Backend API:** https://ragpulse-x7-ai-resume-screening-system.hf.space

## ✨ Features

* Resume PDF Parsing
* Semantic Resume–Job Matching
* Skill Gap Analysis
* Candidate Ranking System
* AI Recruiter Evaluation
* Candidate Comparison
* PDF Report Generation
* Responsive Modern UI
* Cloud Deployment

## 🏗️ Architecture

Resume Upload → Resume Parsing → Embedding Generation → Semantic Matching → Skill Analysis → Candidate Ranking → AI Evaluation → PDF Reporting

## 🚀 Local Setup

### Backend

```bash
cd backend
pip install -r requirements.txt
uvicorn main:app --reload
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```
